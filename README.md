## 📋 System Overview
Founder Compass is a professional-grade Decision Support System (DSS) designed to eliminate the "Black Box" of startup portfolio management.

### Key Features:
1.  **Founder Portal**: guided reporting engine for Weekly Vitals and Monthly Pulses.
2.  **Admin Command Center**: A 9-tab intelligence interface for proactive portfolio management.
3.  **Intelligence Layer**: Automated calculation of Growth Efficiency, Burn Velocity, and Net Margin.
4.  **Security**: Supabase Row-Level Security (RLS) ensuring total data isolation.

## 🛠 Tech Stack
*   **Frontend**: React, Tailwind CSS, Lucide Icons
*   **Database/Auth**: Supabase (PostgreSQL)
*   **Charts**: Recharts
*   **State Management**: TanStack Query (React Query)
